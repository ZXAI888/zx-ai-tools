📢 推广条是什么？
推广条显示在软件界面右上角，是一个小横幅。你可以用它来：
  • 推广你的付费版本或新功能
  • 显示限时优惠活动
  • 引导用户访问特定页面
用户可以点 × 关闭，关闭后下次打开软件不再显示（直到你更新内容）。

📋 公告通知是什么？
公告以弹窗或通知列表的形式出现在软件内。你可以用它来：
  • 发布版本更新说明
  • 通知重要维护信息
  • 推送新功能介绍
支持三种样式：普通（蓝色）、新功能（绿色）、警告（橙色）。

🚀 发布管理是什么？
编辑完推广条或公告后，需要点击「同步发布」才能生效。
发布后用户下次打开软件时会自动获取最新内容，无需用户手动更新软件。
配置只需填写一次，保存后下次直接使用。
